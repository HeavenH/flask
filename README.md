# flask

## Usage

### Linux:

```
cd flask-master
pip install virtualenv
pip install -r requirements.txt
virtualenv venv
virtual source venv/bin/activate
python3 manage.py runserver
```
### Windows:
```
cd flask-master
pip install virtualenv
pip install -r requirements.txt
virtualenv venv
cd venv/scripts
activate
python3 manage.py runserver
```
