# flask

## Usage

### Linux:

```
git clone https://github.com/HeavenH/flask.git
cd flask
pip install virtualenv
pip install -r requirements.txt
virtualenv venv
virtual source venv/bin/activate
python3 manage.py runserver
```
### Windows:
```
git clone https://github.com/HeavenH/flask.git
cd flask
pip install virtualenv
pip install -r requirements.txt
virtualenv venv
cd venv/scripts
activate
manage.py runserver
```
